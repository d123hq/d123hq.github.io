## How it works

The [Pyodide](https://pyodide.org) project compiles CPython to WebAssembly, which all modern browsers can execute natively at near-native speed. The runtime is loaded once from a CDN and then cached by the browser — subsequent page loads are instant.

Limitations to be aware of:

- **No filesystem access** — `open()` works only with in-memory virtual files
- **No subprocess** — can't shell out or spawn processes  
- **Blocking I/O** — network calls require `asyncio` and Pyodide's `pyfetch`
- **Package installs** — run `await micropip.install("package")` for pure-Python wheels
